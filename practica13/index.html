Ejercicio 1 Punto 1.

Los reset en css simplemente ponen a cero todas las propiedades CSS de los estilos que tienen los navegadores por defecto, nos ayuda para mostrar nuestras propiedades css lo más homogéneo posible, es importante saber que esto no hará que los estilos aplicados en css se vean de igual manera en todos los navegadores.

Ejercicio 1 Punto 2

body { margin: 0; padding: 0; }

Podemos observar que al body es decir al cuerpo de la página va a tener el margen a 0 y el padding tambien a 0 anulando así las propiedades por defecto del navegador.

Ejercicio 1 Punto 4 autoprefixer

El autoprefixer es un postprocesador que nos ayuda agregar prefijos de proveedor a las reglas css de nuestra web, donde las verifica en la base de datos de Can i Use para escribir todos los prefijos necesarios de manera que el navegador que usemos lo entienda.
