Explica para qué sirve cada una de ellas.

El estilo Css que hemos aplicado a la etiqueta <a> se aplican cuando el usuario interactua, en este caso hemos puesto:

1º La etiqueta <a> con el valor añadido sea de rojo.

2º Cuando el usuario visite el enlace o la página la etiqueta <a> se pondrá de color azul.

3º La última regla es aplicada cuando el usuario pone el puntero encima del enlace (etiqueta <a>) esto hara que el color de las letras cambie a fuchsia.
