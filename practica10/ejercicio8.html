Ejercicio 8

A partir de las siguientes reglas CSS:

body { font-family: Arial; color: black; }
h1 { font-family: Verdana; }
p { color: red; }

Y del siguiente código HTML:

<body> <h1>Titular de la página</h1> <p>Un párrafo de texto.</p> </body>

Titular de la página

Un párrafo de texto.

¿Se aplica la herencia a algún elemento? Si es así, ¿Qué propiedades se heredan?

Si se aplica una herencia, es en la etiqueta body, los elementos dentro de él, heredan la misma propiedad en este caso la fuente (arial) y el color (negro).
